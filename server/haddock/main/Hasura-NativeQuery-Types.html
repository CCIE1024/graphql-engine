Hasura.NativeQuery.Types

Description

This module houses the types that are necessary to even talk about native queries abstract of a concrete implementation.

The default implementation is given in modules Metadata, and API, but backends are free to provide their own as needed.

So in order to be usable as an API request payload data type, 'TrackNativeQuery b' needs to be wrapped in a newtype.
