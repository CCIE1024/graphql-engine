Description

QuasiQuoted GraphQL constants used througout the codebase. By moving all the Quasiquotes here we can eliminate extraneous rebuilds of larger modules.

See: https://www.parsonsmatt.org/2021/07/12/template_haskell_performance_tips.html#recompilation-avoidance
