type Name = Name 'Column Source #

An alias for Column Names.

This alias is defined in its own module primarily for the convenience of importing it qualified.

For example: @ import Data.Coerce (coerce) import Hasura.Experimental.IR.Column qualified as Column (Name)

example :: Column.Name example = coerce Text Column.Name "column_name" @
