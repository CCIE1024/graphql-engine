Parses some collection of input fields. Build an InputFieldsParser using field, fieldWithDefault, or fieldOptional, combine several together with the Applicative instance, and finish it off using object to turn it into a Parser.

field :: (MonadParse m, 'Input <: k) => Name -> Maybe Description -> Parser origin k m a -> InputFieldsParser origin m a Source #

Creates a parser for an input field. For nullable value parsers, if an input field is omitted, the provided parser will be called with a value of null.

fieldOptional :: (MonadParse m, 'Input <: k) => Name -> Maybe Description -> Parser origin k m a -> InputFieldsParser origin m (Maybe a) Source #

Creates a parser for a nullable field with no default value. If the field is omitted, the provided parser will not be called. This allows a field to distinguish an omitted field from a field supplied with null (which is permitted by the GraphQL specification); see Note [The value of omitted fields] for more details.

If you want a field with a default value of null, use field with a nullable value parser, instead.
